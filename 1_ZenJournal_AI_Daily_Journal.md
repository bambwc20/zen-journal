## 3. 핵심 차별점 기능

### 차별점 1: 실제 작동하는 AI 리플렉션

**Problem (사용자 불만):**
> "AI features are not supported for my Google pixel 9 Pro" — Journal ★1
> "AI feedback/reflection is too generic" — Journal ★3

**Solution:**
기기 의존성 없는 서버사이드 AI 처리 + 사용자 히스토리 컨텍스트 기반 개인화 리플렉션 생성. 단순 감정 키워드 매칭이 아닌, 이전 7일간의 기록 맥락을 포함한 구체적 피드백 제공.

**Implementation:**
- 서버사이드 LLM API 호출 (기기 호환성 문제 원천 차단)
- 최근 7일 일기 컨텍스트 윈도우 적용
- 3가지 리플렉션 타입: 감정 분석 / 패턴 인사이트 / 행동 제안
- 응답 시간 목표: 3초 이내
- 개발 기간: 5일

### 차별점 2: 로컬 암호화 백업 + 내보내기

**Problem (사용자 불만):**
> "please add a way to export or back up the data locally" — Journal ★3
> "not having end-to-end encryption for backup is criminal" — Journal ★1

**Solution:**
기기 내 AES-256 암호화 저장 + 로컬/클라우드 선택형 백업 + PDF/TXT/JSON 내보내기 지원. 사용자가 데이터 소유권을 완전히 보유.

**Implementation:**
- SQLCipher 기반 로컬 DB 암호화
- 자동 백업: Google Drive / iCloud 선택
- 수동 백업: 기기 내 암호화된 .zen 파일 생성
- 내보내기: PDF (포맷 유지), TXT, JSON, CSV
- 개발 기간: 4일

### 차별점 3: 크로스플랫폼 접근성

**Problem (사용자 불만):**
> "I find it hard to journal on a phone, prefer tablet/web version" — Journal ★1

**Solution:**
MVP 단계에서 태블릿 최적화 UI + Progressive Web App(PWA) 베타 제공. 폰에서 간단 기록, 태블릿/웹에서 상세 작성 가능.

**Implementation:**
- Flutter 반응형 레이아웃 (태블릿 2-pane 뷰)
- PWA 기본 뷰어 (읽기 + 간단 작성)
- 클라우드 동기화로 기기 간 실시간 반영
- 개발 기간: 3일 (태블릿 UI), 향후 PWA 확장

## 7. 무료→유료 브릿지 포인트

### 브릿지 1: AI 리플렉션 제한 도달

- **트리거:** 무료 주 2회 AI 리플렉션 소진 시
- **메시지:** "이번 주 AI 인사이트를 모두 사용했어요. 매일 더 깊은 자기 이해를 원하신다면?"
- **전환율 목표:** 8~12%

### 브릿지 2: 7일 스트릭 달성

- **트리거:** 연속 7일 기록 달성 시
- **메시지:** "축하합니다! 7일 연속 기록! Pro로 업그레이드하면 AI 주간 리포트로 이번 주를 돌아볼 수 있어요"
- **전환율 목표:** 5~8%

### 브릿지 3: 검색 제한 도달

- **트리거:** 30일 이전 기록 검색 시도 시
- **메시지:** "이전 기록이 궁금하시군요! Pro에서 전체 기간 검색 + AI 시맨틱 검색을 이용해보세요"
- **전환율 목표:** 6~10%

### 브릿지 4: 내보내기 시도

- **트리거:** 내보내기 메뉴 터치 시
- **메시지:** "소중한 기록을 PDF로 보관하세요. Pro에서 다양한 포맷으로 내보낼 수 있어요"
- **전환율 목표:** 10~15%

### 페이월 디자인

- **타입:** Soft paywall (기능 제한형)
- **무료 체험:** 7일 전체 기능 체험
- **표시 방식:** 바텀시트 모달, 기능별 인라인 잠금 아이콘
- **가격 표시:** 연간 플랜 강조 ($29.99/yr = $2.49/mo 환산 표시)
- **결제 옵션:** 월간 $4.99 / 연간 $29.99 (40% 할인) / 평생 $79.99
